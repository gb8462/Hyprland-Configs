# Hyprland
```bash
sudo pacman -S hyprland 
```

### Setting up
_Clone this repository_
```bash
sudo pacman -S git
git clone https://github.com/gb8462/Hyprland-Configs.git
```

### Using The Script
_This script will automate moving and copying all the configs inside **Hyprland-Configs**_
```bash
cd Hyprland-Configs
./script
```
